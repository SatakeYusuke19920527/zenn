# 記事の作り方
1.以下のコマンドをstart 
   > npx zenn new:article
   > npx zenn new:article --slug 記事のスラッグ --title タイトル --type idea --emoji ✨ (これでもOK)
2. プレビューを確認する
   > npx zenn preview
