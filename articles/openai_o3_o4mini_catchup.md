## はじめに

OpenAI が提供する GPT 系モデルは **GPT-4 系（パターン認識重視）** と **o シリーズ（推論・ツール活用重視）** が主流になっています。  
本記事では、o3 と o4-mini を中心とした以下 7 モデルを対象に **特徴 → 推奨ユースケース → 精度・速度・コスト** の順で整理します。

- **GPT-4o**
- **o3**
- **o4-mini** / **o4-mini-high**
- **GPT-4.1**
- **GPT-4.1 mini**
- **GPT-4.5**（preview）

どの場面でどのモデルを使うべきか、そしてトークン単価（Input / Output）の違いはどうか。迷ったときの参考にこの記事を読んでいただけますと幸いです。

# GPT-o3, o4-mini について

## 概要

o3 と o4-mini は、最先端の推論能力とフル機能のツールセット `ウェブ検索、Python 実行、画像・ファイル解析、画像生成、キャンバス、オートメーション、ファイル検索、メモリ` を組み合わせたモデルです。

複雑な数学問題、コーディング、科学的課題を解くのに優れ、視覚認識・解析力も高い水準を示します。

chains of thought (思考の連鎖)の途中でツールを呼び出し、たとえば画像をトリミング／変換したり、ウェブ検索を行ったり、Python でデータを解析したりして能力を拡張したりも出来ます。

o シリーズは chains of thought (思考の連鎖) を用いた大規模強化学習で訓練されています。

この高度な推論機能は、モデルの安全性と堅牢性を高めます。
特に熟慮型アラインメント により、潜在的に危険なプロンプトに応答する際に、その文脈内で安全ポリシーを推論可能です。

- Preparedness Framework
  - https://cdn.openai.com/pdf/18a02b5d-6b67-4cec-ab64-68cdfbddebcd/preparedness-framework-v2.pdf
  - Preparedness Framework v2（2025 年 4 月 15 日版）— OpenAI が公開した“次世代 AI の危険シナリオを追跡し、安全対策を整えるための社内ルールブック”です。PDF 全 30 ページほどの文書で、OpenAI の Safety & Security チームが策定・運用しています。
  - 将来の「フロンティア・モデル」（既存よりはるかに強力な AI）が社会に大きな害を及ぼす可能性を事前に検知し、展開を止めるか追加の安全策を講じるための枠組みを定義します。

o3 と o4-mini は、OpenAI の「Preparedness Framework」バージョン 2 に基づいて公開される最初のローンチです。OpenAI Safety Advisory Group（SAG）が評価結果を審査したところ、OpenAI o3 と o4-mini は追跡対象の 3 分野（生物・化学能力、サイバーセキュリティ、AI 自己改善）のいずれにおいても High（高リスク）の閾値には達していないと判断されました。

以下ではこれらの評価内容を説明し、リスク軽減に向けた取り組み状況を記載していきます。

# 各モデルの特徴・推奨ユースケース・精度・速度・コスト

## GPT-4o

### 特徴

- テキスト・画像・音声を組み合わせられる **マルチモーダル** 対応
- 音声入力時の平均レイテンシ **≈ 0.3 秒** とほぼリアルタイム
- GPT-4 と同等以上の言語能力を維持しながら **高速 & 低コスト**（従来 GPT-4 の約半額）

### 推奨ユースケース

- 画像や音声を含む **対話アプリ / 音声アシスタント**
- 多言語対応が必須の **翻訳・カスタマーサポート**
- 画像解析・動画要約など **マルチモーダル処理**

### 精度・速度・コスト

| 指標   | 評価                                              |
| ------ | ------------------------------------------------- |
| 精度   | ★★★★★（多言語・視覚入力にも強い）                 |
| 速度   | ★★★★☆（音声で 0.3 秒台）                          |
| コスト | **入力 \$2.5 / 1M tokens・出力 \$10 / 1M tokens** |

---

## o3

### 特徴

- **最上級の推論力**：複雑な数学・コーディング・科学課題で突出
- ChatGPT 上の全ツール（Web 検索・Python・画像解析など）を **自律的に連携**
- 難問での誤答を旧世代から **20 %以上削減**

### 推奨ユースケース

- 競技プログラミング級の **アルゴリズム設計・バグ修正**
- 複雑データの **統計解析 & 可視化**（Python 実行）
- **学術研究・市場調査**など、多段推論＋外部情報取得が必要な課題

### 精度・速度・コスト

| 指標   | 評価                                             |
| ------ | ------------------------------------------------ |
| 精度   | ★★★★★                                            |
| 速度   | ★☆☆☆☆（数秒〜数十秒と遅め）                      |
| コスト | **入力 \$10 / 1M tokens・出力 \$40 / 1M tokens** |

---

## o4-mini / o4-mini-high

### 特徴

- **小型・高速・低コスト**。AIME 数学満点級の実力
- ツール併用で STEM／非 STEM どちらも o3-mini を大幅超え
- **High** モード：推論ステップ増で精度向上（1 日 100 回上限、単価は同一）

### 推奨ユースケース

- 1 日数百件の **技術 Q&A ボット** や **教育チュータ**
- **初歩的データ分析**（Python で統計・プロット）
- **ビジネスレポート要約 / インサイト抽出**（大量処理）

### 精度・速度・コスト

| 指標   | 通常版                                             | High 版  |
| ------ | -------------------------------------------------- | -------- |
| 精度   | ★★★★☆                                              | ★★★★☆ ＋ |
| 速度   | ★★★☆☆                                              | ★★☆☆☆    |
| コスト | **入力 \$1.1 / 1M tokens・出力 \$4.4 / 1M tokens** |

---

## GPT-4.1

### 特徴

- **コーディング性能**が GPT-4o を大幅上回り、SWE-bench 54 %
- **100 万トークン**の超長文コンテキスト
- 指示追従 +10 pt（GPT-4o 比）／知識カットオフ 2024-06

### 推奨ユースケース

- **開発業務支援**（コード生成・リファクタ・レビュー）
- 数百ページ級の **長文ドキュメント解析 / 要約**
- 最新情報を踏まえた **技術レポート作成・チャットボット**

### 精度・速度・コスト

| 指標   | 評価                                           |
| ------ | ---------------------------------------------- |
| 精度   | ★★★★★                                          |
| 速度   | ★★★★☆                                          |
| コスト | **入力 \$2 / 1M tokens・出力 \$8 / 1M tokens** |

---

## GPT-4.1 mini

### 特徴

- GPT-3.5 並みの **超低コスト**で GPT-4 相当の知能
- GPT-4.1 譲りのコーディング・指示追従改善
- ChatGPT 無料枠フォールバックにも採用予定

### 推奨ユースケース

- **大量リクエスト**を高速処理する FAQ / チャットボット
- **文書下読み・要約**のバッチ処理
- 軽量 **プログラミング Q&A**、SNS 感情分析など

### 精度・速度・コスト

| 指標   | 評価                                                 |
| ------ | ---------------------------------------------------- |
| 精度   | ★★★★☆                                                |
| 速度   | ★★★★★                                                |
| コスト | **入力 \$0.40 / 1M tokens・出力 \$1.60 / 1M tokens** |

---

## GPT-4.5（preview）

### 特徴

- GPT-4 系最大規模で **知識網羅性 & 文章表現力** が最高峰
- 共感的対話や **クリエイティブ文書生成** が得意
- 幻覚率減少・事実整合性向上を実現（ただし 2025-07-14 API 廃止予定）

### 推奨ユースケース

- **ライティング**（物語・コピー・ブログ校正）
- **アイデア発想**・ブレインストーミング
- 感情に寄り添う **対話アシスタント**

### 精度・速度・コスト

| 指標   | 評価                                              |
| ------ | ------------------------------------------------- |
| 精度   | ★★★★★                                             |
| 速度   | ★★☆☆☆                                             |
| コスト | **入力 \$75 / 1M tokens・出力 \$150 / 1M tokens** |

## モデル選定の指針

| ユースケース                 | おすすめモデル      | 理由                             |
| ---------------------------- | ------------------- | -------------------------------- |
| 一般コーディング             | **GPT-4.1**         | 高精度・高速・低コスト           |
| 高度アルゴリズム・デバッグ   | **o3**              | 最上級の推論                     |
| カスタマーサポート／FAQ      | **o4-mini**         | 大量問い合わせに対応／コスト良好 |
| 画像＋音声の対話アプリ       | **GPT-4o**          | マルチモーダル & リアルタイム    |
| 長大文書の解析・要約         | **GPT-4.1**         | 100 万トークン対応               |
| ライティング／コピー         | **GPT-4.5**         | 表現力 & 共感力が最良            |
| 教育チュータ（演習解説）     | **o4-mini**         | 数学・論理問題を高速解説         |
| データ分析（コード実行込み） | **o3**              | 自律的にツール連携して解析       |
| 最新情報リサーチ             | **o3**＋**GPT-4.5** | Web 検索 + 知識網羅で補完        |
